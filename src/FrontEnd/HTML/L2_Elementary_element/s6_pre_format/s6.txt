有时候，需要在网页上显示代码，比如java代码

就需要用到pre


示例 1 : 预格式

 <p>这里是没有用预格式的情况：</p>

 public class HelloWorld {

     public static void main(String[] args) {
         System.out.println("Hello World");
     }
 }

 <br/>
 <br/>

 <p>使用预格式的情况:</p>

 <pre>
 public class HelloWorld {

     public static void main(String[] args) {
         System.out.println("Hello World");
     }
 }

 </pre>
